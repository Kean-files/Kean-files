# Kean Files - Photo Gallery

A beautiful, responsive photo gallery website where you can easily add, view, and manage your photos!

## Features

✨ **Easy to Use**
- Add photos by entering a title and image URL
- Delete individual photos
- Clear all photos at once

📱 **Responsive Design**
- Works perfectly on desktop, tablet, and mobile
- Beautiful gradient background
- Smooth animations and transitions

💾 **Persistent Storage**
- Photos are saved in your browser's local storage
- Your gallery persists even after closing the browser

🎨 **Modern UI**
- Clean, modern interface
- Hover effects and smooth animations
- Professional color scheme

## How to Use

1. **Add a Photo**:
   - Enter a title for your photo
   - Enter the URL of an image (must end in .jpg, .png, .gif, or .webp)
   - Click "Add Photo" or press Enter

2. **View Gallery**:
   - Photos appear in a responsive grid layout
   - Hover over cards to see effects

3. **Delete Photos**:
   - Click the "Delete Photo" button on any card
   - Or click "Clear All" to remove all photos

## Getting Started

1. Clone this repository:
   git clone https://github.com/Kean-files/Kean-Files.git

2. Open index.html in your web browser

3. Start adding your photos!

## Image URL Tips

You can use image URLs from:
- **Unsplash**: https://unsplash.com
- **Pexels**: https://pexels.com
- **Pixabay**: https://pixabay.com
- **Your own server** or cloud storage

Just make sure the URL ends with an image extension (.jpg, .png, .gif, or .webp)
